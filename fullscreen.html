  • Our global culture is shaped and made possible by the internet

What is software?

  • Software enables a user to control a computer
  • Each computer has software called the operating system
  • The operating system controls all the other software
  • Other programs might include a text editor or web browser

What is open source software?

  • Programmers write source code to develop software
  • When a user runs a software program the source code is not needed
  • Without the source code changes cannot be made to the software
  • Proprietary software is patented and does not provide source code
  • Open source software programs have source code available to modify or copy

Why do programmers make open source software?

  • The strongest theme is a spirit of collaboration
  • The return is greater than the investment due to a large community
  • Some programmers enjoy writing software or fixing problems
  • Others view proprietary software as unethical

How did open source software begin?

  • In 1952 the users of the first electronic computer by IBM worked together to write software
  • An antitrust case in 1958 forced AT&T to freely license their Unix operating system
  • Unix and other software became closed source and patented starting in 1983
  • Richard Stallman founds the GNU Project in 1984, championing the idea of free software

How is open source related to patents?

  • Open source software is at odds with patent owning corporations
  • The GNU General Public License created by Richard Stallman describes free software
  • It guarantees users freedom to run, study, share and modify software
  • Open source software enhances the ongoing software patent laws argument

How does software affect the world?

  • In our computerized world proprietary software creates problems
  • Modern farm equipment is being sold with proprietary software
  • Equipment cannot be fixed by the owner due to the lack of source code
  • The open source Farm Hack community provides high tech alternatives
  • The UN led Open Foris initiative is empowers developing countries to survey forests
  • "Let's assume that you think you have a choice of eight paths to follow...and let's assume you can't see any real purpose in any of the eight. Then...you must find a ninth path" "

    Hunter S. Thompson

Are there any reasons to not use open source software?

  • Software programs are tools, and each tool has a specific use that it is best for
  • The most common critcism is that it doesn't match the competition
  • Proprietary software is sometimes the best choice
  • Open source software can provide free or cheap alternatives

What was that about the internet?

  • Open source software and the internet have grown together
  • Internet standards and were developed in the spirit of collaboration
  • Without the internet distribution of open source software would be limited
  • The most popular operating system to run web sites is the open source software Linux
  • People, families, communities and the world are connected more closely than ever before by the internet, and without open source software, the internet would be a shadow of what it is today. The free software movement and open source software have helped make computing, communication and collaboration cheaper and more accessible. The ideals of open source software have challenged the existing mindsets and expectations of the world, showing new ways to work together and overcome problems.
